Detailed Description

Class encapsulating all runtime information for each class.

This class is a wrapper for accessing the current runtime instances of the classes OutputChannel and Error. Every autopin class relying on the services provided by these classes gets an own instance of AutopinContext.

See also:
OutputChannel, Error, Configuration
