Detailed Description

This class implements the support for perfmon.

It provides support for the original perfmon monitoring interface. Therefore a patched kernel is needed for this monitor.

The implementation is based on libpfm 3.2 due to the setup of the reference testing system.

Many parts of the class were implemented with the help of task_attach.c from the programming examples included in libpfm 3.2.
